# SEO Command Center

Emika AI Employee — SEO Manager. A comprehensive SEO audit, content optimization, and tracking platform.

## Features

- **Site Audits** — Crawl and analyze websites for SEO issues, track scores over time
- **Content Studio** — Create SEO-optimized content with real-time analysis, SERP preview, keyword density tracking
- **Issue Tracker** — Track all SEO issues with severity, status, priority, and recommended fixes
- **Keywords** — Track keyword positions, volume, difficulty, intent, and clusters
- **Links** — Monitor backlinks, find opportunities, manage outreach campaigns
- **Integrations** — Setup guides for GA4, Search Console, SEMrush, Ahrefs, Moz, WordPress, Shopify, Webflow, Wix, Squarespace
- **AI Engine Optimization (AEO/GEO)** — 2026 strategies for AI search visibility

## Quick Start

```bash
npm install
node server.js
```

Server runs on port 3000. Open http://localhost:3000.

## Stack

- Express.js backend with JSON file storage
- Vanilla JS SPA frontend (no framework dependencies)
- Dark theme UI with DM Sans font

## Data

Stored in `/home/node/emika/seo-hub/` (production) or `./data/` (local development).
